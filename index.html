Who Won?

Description of the Game "Spy Duo"

This game is an adaptation of the classic "Spy" game for two players. The main goal is to figure out if their locations match by asking indirect questions.

Rules

  1. Start of the Game:

    Each player gets one of two random locations, for example:

    • Player 1: "Subway"
    • Player 2: "Hospital"

  2. Objective:

    Determine if the locations match by asking questions.

  3. Game Flow:

    Players take turns asking each other questions. Questions must be indirect, such as:

    • "How often do you visit this place?"
    • "What do you like about this place?"

  4. Direct Question:

    Each player can ask only one direct question during the game. For example:

    • Player 1: "Is it a hospital?"

      If the answer is yes, Player 1 wins.

    • Player 1: "Is it a subway?"

      If the answer is no, Player 1 loses.

Victory

The game ends when one of the players correctly identifies whether the locations match or do not match, or when one of the players successfully guesses the exact location using their one direct question.
